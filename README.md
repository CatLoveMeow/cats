# Casting Board

Casting Board is a personal tool for assigning a face or photo to each character name in a novel you're reading. It helps you keep track of characters and picture them more clearly!

## How it works

- **No Backend, No Build Step:** This application is just a single static HTML file containing everything needed to run.
- **Privacy First:** All data is saved in your browser's local storage (`localStorage`). This means your data is completely private to the browser and device you open the site on. It will not sync across devices or be sent to any server.

## Running Locally

To run the app locally, you have a few simple options:
1. Double-click the `index.html` file to open it in your browser.
2. Or serve the directory using any static web server (e.g., `python -m http.server`, `npx serve`, etc.).

## Enabling GitHub Pages

If GitHub Pages is not yet enabled for this repository, you can turn it on by following these steps:
1. Go to your repository on GitHub.
2. Click on **Settings** (the gear icon near the top right).
3. In the left sidebar, click on **Pages**.
4. Under "Build and deployment", set the **Source** drop-down to "Deploy from a branch".
5. Under "Branch", select your **main** branch and the **/ (root)** folder.
6. Click **Save**.

The site will take a few minutes to build and deploy. Once complete, your site will be live!
